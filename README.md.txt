Excel Sales Performance Dashboard

  Project Overview
This project demonstrates my ability to analyze business data and build an interactive Excel dashboard using Pivot Tables, KPIs, and Slicers.

The dashboard provides insights into sales performance, profitability, and trends to support business decision-making.

  Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Excel formulas
- Dashboard design principles

Key Metrics (KPIs)
- Total Sales
- Total Profit
- Profit Margin %

Dashboard Features
- Interactive slicers for Region, Product Category, and Sales Channel
- Monthly sales trend analysis
- Sales performance by region
- Profit contribution by product category
- Dynamic KPIs linked to slicers

 Business Insights
- Identified top-performing regions and product categories
- Analyzed profitability across sales channels
- Highlighted trends to support strategic planning

 Skills Demonstrated
- Data cleaning and preparation
- MIS reporting
- Data analysis and visualization
- Business insight generation
